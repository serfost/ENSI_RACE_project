## Headings 
- Use # for headings (# H1, ## H2, ### H3)
## Bold/Italic 
- **Deux étoiles "*" de chaque côté : gras**
- *Une de chaqsue côté : italique*
## Lists
- "-" or "*" for unordered, "1." for ordered
## Links
- [text] (URL)
## Images
- ![alt text] (image URL)
## Code Blocks
- Use backticks ( alt gr + 7 ) `code` for inline or triple backticks for blocks

/!\ Les espaces sont importants
